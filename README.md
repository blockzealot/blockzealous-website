# BlockZealous Website

This is the source code for the BlockZealous website, built with React, Vite, and Tailwind CSS.

## How to Use

Since this project is automatically deployed via Vercel, any changes pushed to the main branch will be reflected on the live site.

## Project Structure

- `index.html`: The main HTML file.
- `package.json`: Contains the project dependencies and scripts.
- `vite.config.js`: Configuration for Vite.
- `tailwind.config.js`: Configuration for Tailwind CSS.
- `postcss.config.js`: Configuration for PostCSS.
- `src/`: Contains the React components and CSS files.
  - `main.jsx`: Entry point for the React application.
  - `Website.jsx`: Main website component.
  - `index.css`: Tailwind CSS imports.

## Dependencies

- React
- Vite
- Tailwind CSS
- Lucide React Icons
